

# 简介

本项目时通过autojs实现自动化完成强国任务，目前处于开发阶段。

欢迎大家star，提出意见，你的star时我更新的动力！！

# 使用方式

- 安卓下载Auto.js，（IOS不支持）把代码移动到脚本文件夹
- 点击main脚本运行，确保应用开启无障碍模式

ps：后期会将打包应用上传
