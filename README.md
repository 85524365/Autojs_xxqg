

# 简介

本项目时通过autojs实现自动化完成强国任务，目前处于开发阶段。

欢迎大家star，提出宝贵的意见，你的star是我更新的动力！！

# 使用方式

- 安卓下载Auto.js，（IOS不支持）把代码移动到脚本文件夹
- 点击main脚本运行，确保应用开启无障碍模式

ps：后期会将打包应用上传

# 开发文档

- 本项目使用模块化开发，具体的开发文档可以参考：https://www.frontblog.top/autojs
- 欢迎提出Issues，技术上的探讨。
