# 使用方式

- 安卓下载Auto.js，（IOS不支持）把代码移动到脚本文件夹
- 点击main脚本运行，确保应用开启无障碍模式

# 版本语义化

```
1.2.3 -beta.1 +meta
```

1：代表主版本更新，存在重大更新

2：代表次版本更新，API更新，功能更新

3：代表修订号，修复Bug

beta：代表对外公测版本，存在一定Bug（先行版本）

meta：修改内容

**先行版本**：

alpha：内测版本，存在大量Bug

beta：公测版本，消除严重Bug，仍然存在缺陷

rc：候选版本，不会加入新功能，主要是排错，修改Bug

release：稳定，正式版本
